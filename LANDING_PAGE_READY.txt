🔗 CUSTOMIZATION

To Update Download Links:
  1. Open index.html in text editor
  2. Find "github.com/Ntrospect/websler/releases/latest"
  3. Replace with your actual Windows download URL
  4. Find "testflight.apple.com"
  5. Replace with your actual TestFlight link

To Add Your Logos:
  1. Create /public_html/assets/ folder
  2. Upload websler-logo.png and jumoki-logo.png
  3. Uncomment logo section in HTML
  4. Update image paths

To Change Company Info:
  1. Find footer section (near bottom)
  2. Update email and website links
  3. Update company name if needed

See LANDING_PAGE_DEPLOYMENT.md for detailed customization guide.
